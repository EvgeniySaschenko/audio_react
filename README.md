# WEbpack
*Коменды в консоле:*
* **npm i** - установка модулей
* **npm run server** - запуск проэкта для разработки
* **npm run build** - сборка проэкта для прододакшина


* Что использовалось: React, Redux, React Router, JavaScript, SASS, Webpack, BEM, PHP, MySQL
